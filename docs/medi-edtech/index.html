Overview

Medi-EdTech is an educational tool designed to help students, educators, and healthcare professionals learn about medications interactively. The platform provides a searchable database of medications, including their uses, dosages, side effects, and detailed explanations of how each medication works.

Key Features

  • Searchable Medication Database: Quickly search for medications by name, use, dosage, or side effects.
  • Filter by Medication Type, Dosage, and Side Effects: Refine your search results to find exactly what you need.
  • Detailed Medication Information: Access in-depth information on each medication's use, dosage, side effects, and mechanisms of action ("How It Works").
  • Interactive Interface: Designed with students and educators in mind, the clean and intuitive user interface makes navigation easy and effective.

Tech Stack

This project is built with the following technologies:

  • Frontend: React.js, Vite, CSS
  • Data: JSON file format for medications data
  • UI/UX: Responsive design with mobile-first approach

Installation Instructions

Clone the Repository

To get started, clone the repository:

git clone https://github.com/CatoRaymond-FS/medi-edtech.git

Install Dependencies

Install the necessary dependencies by running:

npm install

Start the Development Server

Start the project in development mode:

npm run dev

The app will be available at http://localhost:5173

Usage

Once the application is running, you can:

  • Search for medications using the search bar.
  • Filter medications based on type, dosage, or side effects.
  • Click on any medication card to view more detailed information about it.
  • Learn how the medication works in the "How It Works" section on each medication's detail page.
