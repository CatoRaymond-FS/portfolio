Procedural Dungeon Generator

The Procedural Dungeon Generator is an AI-assisted web app that dynamically creates connected, explorable dungeon maps in real time. It uses a FastAPI backend with WebSocket streaming for live generation and a React + Three.js frontend for interactive 3D visualization.

  • Real-time dungeon generation through WebSockets
  • Interactive editing and AI entropy tracking
  • FastAPI backend with a React + Three.js frontend

EdTech Backend API

The EdTech Backend API is a Node.js + Express backend for managing users and courses in an educational platform. It uses MongoDB with Mongoose for data modeling and includes a built-in static frontend tester for experimenting with API calls.

  • RESTful API for user and course management
  • Frontend tester UI for interactive API testing
  • Clean folder structure for scalability
  • Future-ready: JWT auth, admin dashboards, progress tracking
