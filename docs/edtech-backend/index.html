Overview

The EdTech Backend is a RESTful API that supports an educational technology platform, managing users, courses, and interactions. It is designed to work with a lightweight frontend tester and can be extended for full-scale deployment.

Key Features

  • User Management: Add, retrieve, update, and delete users.
  • Course Management: Create and fetch courses with key data.
  • Frontend Tester: Interact with the API using a user-friendly interface.
  • Modular Design: Easily extendable via Express routes and Mongoose models.

Tech Stack

  • Backend: Node.js, Express.js
  • Database: MongoDB with Mongoose
  • Frontend Tester: HTML, CSS, JavaScript (static UI)
  • Deployment: Render (Backend), GitHub Pages (Frontend)

Installation Instructions

1. Clone the Repository

git clone https://github.com/CatoRaymond-FS/edtech-backend.git

2. Install Dependencies

npm install

3. Configure Environment Variables

Create a .env file in the root directory and add:

MONGO_URI=your-mongodb-connection-string
PORT=3000

4. Start the Server

node server.js

Visit http://localhost:3000 to access the API.

API Endpoints

  • GET /api/users – Fetch all users
  • POST /api/users – Add a new user
  • GET /api/courses – Fetch all courses
  • POST /api/courses – Add a new course
